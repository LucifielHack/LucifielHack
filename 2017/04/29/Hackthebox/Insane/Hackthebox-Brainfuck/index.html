Http

直接访问是 Nginx 默认页面

使用 hosts 解析后访问是一个 wordpress 页面，版本是 4.7.3

这里的内容是 SMTP 集成已准备就绪吗，然后给了我们一个邮件地址，让我们检查后把反馈发送到这个地址

orestis@brainfuck.htb

SMTP 使用的端口是 25 端口，这里先记录一下，说不定有用
